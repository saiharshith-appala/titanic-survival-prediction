# Titanic Survival Prediction

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning algorithms.

The project uses passenger information such as:
- Passenger Class
- Gender
- Age
- Fare
- Embarked Location

to predict survival status.

Output:
- 1 = Survived
- 0 = Not Survived
